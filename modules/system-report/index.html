    System Report¶

    Generate cluster, VM, node and storage reports.

    Overview¶

    Generates reports on Proxmox VE infrastructure including clusters, nodes, virtual machines, and storage resources.

    • Infrastructure Reports

      Generate reports for compliance audits and system documentation.

    • Resource Data

      View resource utilization data for capacity planning.

    • Configuration Data

      Document system configurations, VM specifications, and storage allocations.

    • Report Generation

      Generate reports in multiple formats with on-demand or scheduled execution using cron expression.

    • Proxmox VE Integration

      Uses Proxmox VE API to collect system information.

    • Report Formats

      Export reports in structured formats for archival and sharing.
